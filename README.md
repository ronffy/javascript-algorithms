## 算法的时间和空间复杂度

评价一个算法的效率主要看该算法的时间复杂度和空间复杂度。  

时间复杂度：执行算法所消耗的时间

空间复杂度：执行算法需要占用多少内存空间

### 大O符号表示法

无论是描述时间复杂度还是空间复杂度，最常用的是使用 大O符号表示法 。大O符号表示法并不是用来表示算法的真实执行时间的，它是用来表示代码执行时间的增长变化趋势的。

常见的复杂度量级有（自上至下复杂度越来越大）：
符号 | 描述
---- | ----
O(1) | 常数阶
O(logN) | 对数阶
O(n) | 线性阶
O(nlogN) | 线性对数阶
O(n²) | 平方阶
O(n³) | 立方阶
O(n^k) | K次方阶
O(2^n) | 指数阶
